# Mon Projet

Ce projet est une application web développée dans le cadre d'un cours à Ynov Campus. Il s'agit d'une application de gestion de profil utilisateur qui permet aux utilisateurs de créer un compte, de se connecter et de gérer leur profil.

## Fonctionnalités

- **Inscription et Connexion :** Les utilisateurs peuvent créer un compte en fournissant leur adresse e-mail, leur mot de passe et leur nom. Ils peuvent également se connecter à leur compte existant.
- **Gestion de Profil :** Une fois connectés, les utilisateurs peuvent voir leur profil et mettre à jour leurs informations, y compris leur nom et leur photo de profil.
- **Upload d'Image :** Les utilisateurs peuvent choisir une image de leur galerie pour l'utiliser comme photo de profil.
- **Accessibilité :** L'application est accessible à l'adresse suivante : [https://github.com/Sinclqir/web-cloud-ynov/](https://github.com/Sinclqir/web-cloud-ynov).

## Technologies Utilisées

- **Frontend :** React, React Native Paper, Expo Image Picker
- **Backend :** Firebase (Authentification, Stockage)

## Comment Utiliser

1. Clonez ce dépôt sur votre machine locale.
2. Installez les dépendances en utilisant `npm install`.
3. Lancez l'application en utilisant `npm start`.
4. Accédez à l'application dans votre navigateur à l'adresse [http://localhost:8081](http://localhost:8081).
